### 1.2 计算公式

```typescript

calcStaminaRecovery(physique) = min(500, 200 + 3 * physique)

maxLifespan = floor(60 + physique * 0.4 + fortune * 0.2)
```

## 三、伤害事件系统

### 3.1 随机事件伤害触发

仅在属性 < 30 且选择事件的第一个选项时触发：

| 属性值 | 死亡概率 | 重伤概率 | 轻伤概率 |
|--------|----------|----------|
| < 10 | 10% | 25% | 40% |
| 10-20 | 5% | 15% | 25% |
| 20-30 | 2% | 8% | 15% |

### 3.2 伤害效果

- **轻伤**：所有属性 -5%
- **重伤**：所有属性 -20%
- **死亡**：直接结束游戏

### 3.3 触发机制说明

- 在随机事件（EventModal）中，选择**第一个选项**（idx=0）时视为"迎难而上"类型的选择，可能触发伤害
- 例如"拔剑迎战，以武制暴"、"揭榜献策，一展抱负"等积极/冒险选项
- 探索页面：武艺 < 30 时有 35% 概率触发打劫伤害（无需选择，自动触发）

calcShopProfit(basePrice, generationGap) = floor((pow(3 * generationGap, 2) + 1) * basePrice)

## 六、年终结算分数计算

```typescript
最终总分 = 声望分 + 财富分 + 天赋技艺分 + 科举加分 + 官职加分 + 人脉产业分 + 成就称谓分

其中：
- 声望分 = floor(prestige / 10)
- 财富分 = floor(wealth / 20)
- 天赋技艺分 = knowledge + martial + craft + strategy + charm
- 科举加分：秀才50, 举人150, 解元200, 贡士300, 会元400, 进士500, 探花600, 榜眼700, 状元1000
- 官职加分 = max(0, (10 - officialRank) * 100)  （注意：此处officialRank越小品级越高）
- 人脉产业分 = friends.length * 10 + lands.length * 20
- 成就称谓分 = 各成就finalScore之和
```

## 八、成就/称谓系统

### 8.1 系统验证结果

成就称谓系统已在游戏中完整实现：
- 解锁检测：[achievements.ts:545-619](src/mocks/achievements.ts#L545-L619) 的 `checkAchievementUnlock` 函数
- 加成应用：在 [game/page.tsx:368-376](src/pages/game/page.tsx#L368-L376) 中，成就解锁时会立即将 `bonuses` 中的属性加成加到角色上
- 每年金钱加成：[game/page.tsx:531-536](src/pages/game/page.tsx#L531-L536) 的 `calcAnnualWealthBonus` 函数会计算所有已解锁成就的 `annualWealthBonus` 之和
- 最终结算加分：[achievements.ts:644-698](src/mocks/achievements.ts#L644-L698) 的 `calcFinalScore` 函数会累加所有成就的 `finalScore`

### 8.2 成就结构

```typescript
{
  id: string,                         // 成就ID，如 "ach-new-achievement"
  title: string,                      // 称谓名称
  desc: string,                      // 描述
  category: string,                   // 分类（科技/科举/财富/声望/武艺/学识/官职/长寿/社交/综合）
  icon: string,                      // Remix Icon类名
  color: string,                     // 颜色
  condition: {
    type: "tech"|"exam"|"prestige"|"wealth"|"knowledge"|"martial"|"craft"|"strategy"|"age"|"friends"|"lands"|"official"|"custom",
    value?: number,               // 数值条件
    techId?: number,              // 科技ID（type="tech"）
    examRank?: string,           // 科举功名（type="exam"）
    officialRank?: number,            // 官职品级（type="official"）
    customKey?: string            // 自定义条件标识（type="custom"）
  },
  bonuses: {                      // 永久属性加成（解锁时立即生效）
    prestige?: number,
    knowledge?: number,
    martial?: number,
    craft?: number,
    strategy?: number,
    charm?: number,
    physique?: number,
    wisdom?: number,
    fortune?: number,
    maxLifespan?: number,
    examBonus?: number,             // 科举答题加成（分）
    annualWealthBonus?: number,     // 每年金钱加成（两/年）
    staminaBonus?: number          // 体力上限加成
  },
  finalScore: number               // 结算加分（游戏结束时计入总分）
}
```
