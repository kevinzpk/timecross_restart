## ✨ 核心爽点玩法
真实宏大的历史画卷​：游戏构建了从秦汉到明清的完整历史观。你将亲历王朝更迭，体验真实的历史事件。你的科技树、知识储备甚至你的一言一行，都将与这个时代深度绑定。
深度角色养成系统​：人物拥有声望、金钱、体魄、悟性、气运、魅力等十大基础与技艺属性，以及隐藏的寿命属性。你的每一次选择、每一次学习、每一次冒险，都将真实地改变你的属性，决定你最终的命运。属性面板直观清晰，成长路径自由多样。
跨时代科技树​：你将带着前世的职业记忆穿越。是工程师？是化工专家？还是军事天才？你的职业将直接解锁相应类别的科技线。游戏内置了横跨上古到近现代的 60+项 精细科技，从阉猪养殖到无烟火药，从水利工程到金融管理，科技树有清晰的前置依赖和跨代研发惩罚，策略性极强。研发出的科技不仅能解锁商品，更能用于生产经营，领先时代，富甲一方。
每一条都是穿越文最顶的爽点，做成可玩的交互：
1. **🎲 开局随机投胎**：抽取「前世身份」，不同职业自带专属技能树：工程师解锁机械/化工科技更快，农业天生农田加成buff，金融商业加成拉满😏
2. **📜 科举从政高升**：从院试考到殿试，用现代逻辑学断奇案、用现代管理学理民生，从七品县令一路做到当朝一品，位极人臣 just so easy
3. **⚙️ 科技降维打击**：从蒸馏酒、玻璃器皿、黑火药一路研发到简易蒸汽机，跨代研发有真实惩罚，真·靠你脑子里的现代知识碾压古代，没有无脑金手指
4. **⚔️ 征战开疆拓土**：募兵、训练、打造新军，给军队配上火药火铳，打匈奴、抗倭寇，还能带队一路平推日本岛，封狼居胥不是梦
5. **👑 权谋篡位称帝**：攒人脉、收党羽，玩一出「指鹿为马」试探人心，进度满100%直接黄袍加身，从权臣到开国皇帝，自己开新朝代
6. **🎡 休闲花样百出**：
- 逛花楼卖诗：抄错本朝已有诗词直接声望暴跌身败名裂，全靠你肚里的存货吃饭
- 拍卖行淘宝捡漏，拍到奇珍直接永久加属性
- 结交李白、苏轼、包拯这些历史名人，好感刷满就能拜师学艺，甚至娶回家生娃
- 培养子女天赋，子嗣成年反哺你的属性，打造百年望族，家族声望代代传承
7. **🗺️ 名城探索奇遇​**：解锁长安、汴京、金陵、扬州等各朝历史名城，每城独有专属隐藏剧情，帮包拯断案、和李白对饮、随郑和下西洋，触发奇遇拿顶级属性🤩

## 新手实用攻略
**游戏攻略**： https://timecross.quzhi.life/guide
1. **开局roll点技巧​：**
  - 想快速体验爽感选「前世工程师」→科技线发育最快，成型早
  - 想玩称帝/从政线选「前世律师/公务员」→从政声望加成高，晋升速度快
  - 想玩经商线选「前世商科/销售」→买卖利润加成更高，攒钱速度翻倍
2. **通用发育技巧**​：前期优先点满「学识」属性，参加科举入仕免除税负；中期优先攒声望，声望达标才能解锁高阶官职、开大型商铺、招募更多军队；多探索城市触发奇遇，顶级属性加成大多来自隐藏剧情。
3. **各路线通关要点**​：
  - 科技流：不要一开始就研发跨太多代的科技，惩罚会极高，先从低门槛的蒸馏酒、玻璃做起赚第一桶金，再逐步点高等级科技
  - 科举流：断案优先选「逻辑推导」选项，比刑讯逼供获得的声望高30%，中期多刷京官好感，人脉够了能直接加快晋升速度
  - 称帝流：篡位进度到80%以后不要急着发动，先把京城禁军首领好感刷满，政变成功率直接提升30%
  - 家族流：子女出生先看天赋，文天赋送科举、武天赋扔军队，成年子女高属性会直接反哺玩家全属性，比自己刷属性效率高很多

## 🛠️ 我的AI辅助开发流程
个人独立做游戏，全靠AI提效，流程分享给大家：
1. **第一步：AI梳理游戏策划**：核心玩法、细节设定全部和AI对话梳理，一天就把完整策划案捋清楚，省了大把写文档的时间
2. **第二步：AI生成前端代码**：用 [Readdy.ai](https://readdy.ai/) 输入设计需求，直接生成完整可运行的前端页面和代码，一键导出
3. **第三步：本地深度开发**：把AI生成的代码导入vscode，然后通过claude code进行深度开发，补全后端逻辑、交互细节、玩法功能，一个人就能搞定完整开发

原来做游戏要凑前端、设计、策划，现在用AI工具，一个人就能快速推进自己的游戏想法👍
